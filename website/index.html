03 · projects

Featured builds

AWSMonitoringIAM

EC2 Monitoring with CloudWatch & SNS

Deployed an Ubuntu EC2 instance running a live Nginx web server, with the CloudWatch Agent collecting real CPU and memory metrics. Configured an IAM Role scoped to CloudWatchAgentServerPolicy so the agent never touches access keys, built a CloudWatch Dashboard, and wired a CloudWatch Alarm to an SNS topic that emails an alert the moment CPU crosses 50%. Validated the full pipeline by stress-testing the instance and confirming the alert actually fired.

Static Website on S3 + CloudFront

Hosted a static website on Amazon S3 and distributed it globally through Amazon CloudFront for low-latency delivery, caching, and HTTPS. Focused on getting the access model right — origin access control rather than a public bucket — so the site is reachable without exposing storage directly.
